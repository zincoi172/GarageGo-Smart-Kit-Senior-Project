## How to Run

1. Flash the `.cpp` code inside `code/` folder using **Arduino IDE**.
2. Connect Blynk app with the token and map controls to `V3` (Relay), `V1`, `V0`, etc.
3. Power up the hardware and control the garage from your phone.
